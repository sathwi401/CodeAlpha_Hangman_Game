# 🎮 Hangman Game

## 📌 Project Overview

This is a simple text-based Hangman Game developed in Python as part of the CodeAlpha Python Programming Internship.

The player has to guess a randomly selected word one letter at a time before running out of chances.

## ✨ Features

* Random word selection
* 5 predefined words
* Maximum of 6 incorrect guesses
* ASCII Hangman graphics
* Input validation
* Replay option

## 🛠 Technologies Used

* Python 3

## 📚 Python Concepts

* Functions
* Loops
* Conditional statements
* Lists
* Strings
* Random module
* User input

## ▶️ How to Run

1. Download the Python file.
2. Open a terminal.
3. Run:

python task1_hangman.py
